# Examples

Compile any `.bas` file with FreeBASIC. Each example uses:

    #include once "../vt/vt.bi"

This assumes the `vt` folder is installed in FreeBASIC's `inc` directory.
See the main README for installation instructions.

**Windows users:** place `SDL2.dll` in this `examples/` folder before running
any compiled executable. The DLL must sit alongside the `.exe`, not in the
project root.

Any external assets used by the examples (font sheets, .vts files) are in
the `resources/` subfolder.
